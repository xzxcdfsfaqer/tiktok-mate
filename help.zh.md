## 概述

**TikTok Mate** 是一款浏览器扩展，旨在增强你的 TikTok 使用体验。它能在侧边栏中实时展示视频数据分析、支持一键无水印下载、AI 智能内容分析以及自动浏览等功能。

## 快速开始

1. 安装扩展并将其固定到工具栏。
2. 打开任意 TikTok 视频页面（`http://www.tiktok.com/`）。
3. 点击扩展图标打开侧边栏。
4. 侧边栏会自动显示当前视频的数据。

> **提示：** 如果数据未显示，请尝试刷新 TikTok 页面。

## 功能介绍

### 数据面板

打开 TikTok 视频后，侧边栏将展示以下数据：

| 指标 | 说明 |
|------|------|
| 👁️ 播放量 | 总播放次数 |
| 🏆 质量分 | TikTok 官方画质评分（0-100），影响系统推荐 |
| 📊 互动率 | （点赞+收藏+评论+转发+分享）÷ 播放量 |
| ❤️ 点赞 | 点赞数及占比 |
| 🔖 收藏 | 收藏数及占比 |
| 💬 评论 | 评论数及占比 |
| 📤 分享 | 分享数及占比 |
| 🔁 转发 | 转发数及占比 |

其他信息：视频 ID、作者、发布时间、时长、语言、标签。

点击下载按钮即可将内容保存到本地：

| 按钮 | 功能 |

| 🖼️ 下载图片 | 下载图文帖的所有图片（JPG） |

| 按钮 | 功能 |

| 📋 AI 内容总结 | 结构化内容策略与互动分析 |

> **工作原理：** 点击 AI 按钮将自动打开 DeepSeek Chat 新标签页，并填入视频数据与专业提示词。

### 自动浏览

模拟自然浏览行为，用于账号养号或内容调研。

**设置项：**

| 设置 | 说明 |
|------|------|
| 总数 | 要浏览的视频数量 |
| 速度 | 操作速度，慢速更接近真人行为 |
| 关键词筛选 | 按关键词筛选视频（多关键词用空格分隔） |
| 广告筛选 | 广告视频处理方式：包含 / 忽略 / 仅广告 |
| 操作 | 点赞、收藏、关注 — 可单独开关 |

**使用方法：**

1. 在操作面板中点击 **自动浏览**。
2. 在弹出的对话框中配置参数。
3. 点击 **立刻开始** — 扩展将自动浏览视频。
4. 点击 **点击停止** 结束浏览。

> **建议：** 使用慢速并开启点赞/收藏/关注，养号效果更自然。

## 常见问题

### Q: 侧边栏没有显示数据？

请确保你正在 TikTok 视频页面（`http://www.tiktok.com/`）。如果仍无数据，尝试刷新页面。

### Q: 下载失败或不开始？

下载链接由第三方服务生成。网络问题或服务不可用可能导致失败，请稍后重试。

### Q: AI 分析会发送哪些数据？

仅发送当前视频的公开元数据（播放量、点赞数、描述、评论等）到 DeepSeek Chat。不会传输任何个人信息。

### Q: 自动浏览是如何工作的？

扩展会在你当前打开的 TikTok 页面上模拟用户交互——滚动、观看视频，并可选择点赞、收藏和关注。

### Q: 我的数据安全吗？

安全。所有处理在本地完成。视频数据仅在你主动触发操作（下载、AI 分析）时才会传输到第三方服务。
