# TikTok Mate 账号管理

## 适用场景
- 注册新的 TikTok 账号
- 查看当前哪些账号在线/离线
- 检查指定账号是否在线
- 多账号环境下指定用哪个账号执行操作
- 排查账号不在线的问题

## 工作机制

账号通过浏览器扩展自动保持在线状态：

- **自动注册**: 浏览器扩展打开 TikTok 页面后，账号会自动注册到系统
- **在线保活**: 浏览器扩展每秒自动发送信号，保持账号在线
- **自动离线**: 超过 30 秒未收到信号，系统自动将该账号标记为离线（数据保留，不参与操作分配）

## 流程步骤

### 1. 查看当前在线账号

调用 `alive` 功能查看所有在线账号：

```
alive()
```

返回信息中包含：
- `availableAccounts` — 当前在线的账号列表
- 每个账号包含名称、注册时间和最后活跃时间

### 2. 检查指定账号是否在线

查看返回的在线列表是否包含目标账号：
- **在线**: 列表中能找到该账号，且最近 30 秒内有活跃记录
- **离线**: 列表中找不到该账号，或最后活跃时间已超过 30 秒

### 3. 注册新账号

无需手动注册。只需在浏览器中打开对应 TikTok 账号的页面，确保浏览器扩展已启动，系统会自动完成注册。

### 4. 多账号操作分配

当有多个在线账号时，可在发起操作时指定用哪个账号执行：

```
startJob(jobName="getAccountInfo", jobAccount="userA")
```

## 故障排查

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 账号不在线 | 浏览器扩展未启动或网络不通 | 确认已打开对应账号的 TikTok 页面 |
| 账号反复离线 | 网络不稳定 | 检查网络连接是否正常 |
| 操作未分配到期望账号 | 目标账号不在线 | 先用 `alive` 确认账号在线状态 |
