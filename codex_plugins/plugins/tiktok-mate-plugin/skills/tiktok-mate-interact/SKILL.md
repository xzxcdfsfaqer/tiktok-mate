# TikTok Mate 互动操作

## 适用场景
- 对当前视频点赞
- 收藏当前视频
- 关注当前视频创作者
- 按顺序浏览 Feed 并执行互动（点赞+收藏+关注）
- 批量自动化互动

## 可用互动功能

| 功能 | 说明 | 前置条件 |
|------|------|----------|
| `hitLike` | 点赞当前视频 | 已进入包含视频的页面 |
| `hitFav` | 收藏当前视频 | 已进入包含视频的页面 |
| `hitFollow` | 关注当前创作者 | 已进入包含视频的页面 |
| `gotoNext` | 切换到下一个视频 | 已在 Feed 流中 |

## 流程步骤

### 1. 单次互动操作

**对当前视频点赞**：
```
1. 进入 Feed 页 → 等待完成
2. 点赞当前视频 → 等待完成
```

**点赞 + 收藏 + 关注（组合操作）**：
```
1. 进入 Feed 页
2. 点赞
3. 收藏
4. 关注创作者
```

> 注意: 每个操作需要等待上一个完成后再执行下一个。

### 2. 批量 Feed 互动

按顺序浏览 Feed 并对每个视频执行互动：

```
1. 进入 Feed 页 → 等待完成
2. 循环（每轮对一个视频执行操作）:
   a. 点赞当前视频 → 等待完成
   b. 切换到下一个视频 → 等待完成
   c. 回到步骤 a（或按需收藏/关注）
```

### 3. 互动流水线（按需组合）

```
进入 Feed → 对视频1 点赞 → 收藏 → 下一个 → 对视频2 点赞 → 关注 → 下一个 → ...
```

## 最佳实践

1. **操作间隔**: 每次互动后等待 1-2 秒，避免操作过快
2. **切换方式**: 用「下一个视频」切换，不要重复进入 Feed
3. **频率控制**: 避免过于频繁的操作触发 TikTok 限制
4. **确认结果**: 每次操作后确认是否执行成功
5. **关注去重**: 记录已关注的创作者，避免重复关注

## 故障排查

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 点赞失败 | 已点过赞或页面异常 | 查看错误信息，切换到下一个视频后重试 |
| 关注失败 | 已关注或账号受限 | 检查是否已关注，跳过已关注的创作者 |
| 操作无响应 | 页面状态不对 | 重新进入 Feed 刷新页面状态 |
