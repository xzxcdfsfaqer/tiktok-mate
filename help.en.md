## Overview

**TikTok Mate** is a browser extension that enhances your TikTok experience. It displays real-time video analytics in a side panel, supports one-click downloads (no watermark), AI-powered content analysis, and automated browsing.

---

## Getting Started

1. Install the extension and pin it to your toolbar.
2. Open any TikTok video page.
3. Click the extension icon to open the side panel.
4. The panel automatically displays the current video's data.

> **Tip:** If data doesn't appear, try refreshing the TikTok page.

## Features

### Data Panel

When you open a TikTok video, the side panel shows:

| Stat | Description |
|------|-------------|
| 👁️ Plays | Total view count |
| 🏆 VQScore | TikTok's official quality score (0-100), affects recommendation |
| 📊 Engagement | (Likes + Favs + Comments + Reposts + Shares) ÷ Plays |
| ❤️ Likes | Like count and ratio |
| 🔖 Favs | Favorite count and ratio |
| 💬 Comments | Comment count and ratio |
| 📤 Shares | Share count and ratio |
| 🔁 Reposts | Repost count and ratio |

Additional fields: Video ID, Author, Published Time, Duration, Language, Hashtags.

### Auto Browse

Simulate natural browsing behavior for account nurturing or content research.

**Settings:**

| Setting | Description |
|---------|-------------|
| Count | Number of videos to browse (max 100) |
| Speed | Operation speed — slower is more human-like |
| Keyword | Filter videos by keyword (space-separated for multiple keywords) |
| Ad Strategy | How to handle ad videos: Include / Ignore / Ads Only |
| Actions | Like, Favorite, Follow — toggle each on/off |

**Usage:**

1. Click **Auto Browse** in the Actions panel.
2. Configure your settings in the dialog.
3. Click **Start Now** — the extension will automatically browse videos.
4. Click **Click to Stop** to end the session.

> **Tip:** Use slow speed and enable Like/Favorite/Follow for more natural account nurturing.

## FAQ

### Q: Why doesn't the panel show any data?

Make sure you are on a TikTok video page (`https://www.tiktok.com/`). Try refreshing the page if data still doesn't appear.

### Q: Downloads fail or don't start?

Download links are generated by a third-party service. Network issues or service availability may cause failures. Try again later.

### Q: What data is sent to AI services?

Only the current video's public metadata (plays, likes, description, comments, etc.) is sent to DeepSeek Chat. No personal information is transmitted.

### Q: How does Auto Browse work?

The extension simulates user interactions on the TikTok page — scrolling, watching videos, and optionally liking, favoriting, and following. It runs in the current tab you have open on TikTok.

### Q: Is my data safe?

Yes. All processing happens locally. Video data is only transmitted to third-party services when you explicitly trigger an action (download, AI analysis).
